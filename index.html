How to Play

  • TAB: Toggle SURFACE / NEST view
  • Space: Pause/Resume
  • N: Step one tick
  • R: Reset simulation
  • F: Spawn food cluster near cursor (surface)
  • G: Toggle auto-dig
  • C: Force small chamber at active dig front
  • T: Toggle to-food pheromone overlay
  • L: Toggle to-home pheromone overlay
  • V: Toggle SCENT overlay (food+home)
  • J: Toggle ant JOB color visualization
  • O: Add +50 nutrition to nest FoodStore (cheat)
  • K: Set selected ant hunger low (cheat)
  • D: Toggle danger pheromone overlay
  • Y: Download colony stats log as JSONL (Shift+Y = CSV)
  • F3: Toggle debug overlay (ant H/HP, carry, FoodStore)
  • Left drag: paint tool
  • Right drag / Shift+drag: pan camera (active view only)
  • Wheel: zoom camera (active view only)
